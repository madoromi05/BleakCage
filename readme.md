## ゲーム概要

このゲームは **スキルカード選択式のターン制RPG** です。  
制限時間以内になるべく多くのスキルカードを選択して、大ダメージを狙いましょう。

---

## 操作方法

- **Select画面**  
  - 上下キー：移動  
  - Enter：決定  

- **カード選択**  
  - 1 / 2 / 3  

- **カード決定**  
  - Enter  

- **ガードボタン**  
  - 1 / 2 / 3 / 4  
  - 自キャラが配置されている番号を押す  

- **チュートリアルの説明文**  
  - 右クリックで進む

## インストール・アンインストール方法

### インストール方法

このゲームフォルダ全体を、ご自身の任意フォルダに展開 / コピーしてください。

### アンインストール方法

インストール時に選んだ、ご自身の任意のフォルダ内にある  
このゲームフォルダ全体を削除してください。

---

## 起動方法・終了方法

### 起動方法

インストールしたゲームフォルダ内の  
**「BleakCage..exe」** を起動してください。

### 終了方法

- ゲーム内 Home 画面の「Quit」を左クリック  
- または  
  - タスクキル  
  - タスクバー上から「ウィンドウを閉じる」
